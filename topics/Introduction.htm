Introduction

ChameleonCoder, or short CC is a great, rich-featured programming and scripting IDE including resource management, a rich extensibility model and a lot more...

To get a quick overview about the features see the feature list below. For more information see the concepts section and the QuickStart page.

If you have any suggestions, detected any bug, want to contribute to this project or wrote a new plugin: Don't hesitate to contact me. I'm always open to suggestions and also happy about co-developers.

For the latest updates and nightly builds, to participate on development or for a list of available plugins see the github repo. You may also be interested in the credits.


Features:

  • Rich extensibility model
    • Language Modules
    • Services
    • Templates
    • custom resource types
    • custom RichContent member
  • parsing of resource files and displaying them, including hierarchy
  • 6 included resource types
    • files
    • source code files
    • libraries
    • projects
    • tasks
    • groups
  • user-defined metadata for a resource
  • a "breadcrumb" control for easy navigation
  • creation, moving, copying and deleting of resources
  • multiple language support
  • the possibility to reference a resource in another place

  • ----------------------------------
  • support for coding languages using LanguageModules, which can allow
    • compilation of resources
    • execution of resources
    • syntax highlighting
    • intellisense
    • ... and a lot more
  • of course rich editing component using the AvalonEdit control
  • RichContent model for describing the content of a resource
  • Generating documentation from RichContent, maybe exporting to HTML
  • creation of empty files
  • creation and modification of file references and file metadata
