QuickStart

This page will give you a quick overview about CC's features and user interface.

Opening a file

To open a file, you can either double-click a *.ccr file in explorer, or you can pass it to CC via the command line:

The welcome page

When you opened a file, you will see the welcome page as in the picture.

To the top, you can see the ribbon. The possibilities it gives you now are rather limited, except for the menu:

To the left, you can see a treeview with a hierarchical list of all resources found in the file.
The first image on each item signalizes the resource type, for example libraries use a star and group use a box. If there's a second picture, it displays information specific to the resource type, for example a project's priority (as in the picture).

To the right, you can see the welcome page. It gives you several possibilities:

  • go to the resource list
  • change the configuration
  • view plugins
  • create a new resource


The resource list

The resource list displays a (flat) list of all resources. It displays their name and description and it is grouped by resource type. The icons you can see are the icons of the language modules associated with the resources.

If you want, you can disable grouping and you can filter the resources by resource type to get a better overview in case there are a lot of resources. Resolvable resources arehidden by default. To show them, enable the "Show links" button.


Viewing a resource

When you open a resource by double-clicking it either in the treeview or in the list, you get a view that displays the resource's properties and its metadata.

To the top of the tab, you can see a listview with the resource's properties. You can edit some of them, but other's are readonly or can be set in another way, for example through a button in the ribbon.

On the bottom left, there's a listview containing the resource's metadata. You can add or delete them through the ribbon.

On the bottom right, there's an edit box containing your notes about the data.
