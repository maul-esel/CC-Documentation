Extensibility

ChameleonCoder has a rich extensibility model. This page discusses how you can get plugins and how to handle them.
If you're interested in developing your own, see the Developers section. In case you do so: contact me so that I can add your plugin to the official list.

If you have any problems with a a plugin, please try first contacting the plugin developer. On the website there's a contact address for each plugin.
When that fails you can contact me so that I can look into your problem.

Download and installation

Currently there are 5 ways to extend CC's functionality:

  • LanguageModules
  • Services
  • custom resource types
  • custom RichContent members
  • templates

These are usually located in *.dll files. You can find and download them on the ChameleonCoder website.
Installation is the most easy thing you've ever seen: Just put them into the \Components\ subfolder of the ChameleonCoder directory. Run CC, go to the "plugins" page, and click "install". Select the DLL and select the included plugins you want to install. Restart CC.
To uninstall a plugin, go to the plugins page, select the plugin, and click "uninstall selected".
